-is a function
-that runs between receiving a request and sending a response
-it decides  whether to continue or stop the request
- every middleware function has 3 parameter: 
    (req, res, next)
    parameters       meaning
    req              incoming request 
    res              response we send back 
    next             function to move to next step

- function middleware(req, res, next){
    //code 
    next()  //pass control to next middleware
